## Phase 5 update

`icon.png` (256x256) and `icon.ico` (256x256, PNG-in-ICO) were added as
solid-color placeholders, generated by `desktop/scripts/make-placeholder-icons.js`.
**Not real artwork** — replace before shipping.

`icon.icns` was intentionally NOT generated — producing a valid `.icns`
requires macOS's `iconutil` (or a third-party encoder) which isn't available
in this Windows dev environment. macOS packaging (`pnpm run package:mac` /
the `dmg` target in `electron-builder.config.js`) will fail or fall back
until a real `icon.icns` is added.
